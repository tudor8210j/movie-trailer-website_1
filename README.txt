QuickStart:

In order to install this program first clone this program from GitHub at https://github.com/tudor8210j.
Once cloned you can run the movie_trailer_website_entertainment_center.py file to open the web page.
Do this by either opening it via the Python Shell and running it, running it directly from the Python shell or by double clicking on the file.
